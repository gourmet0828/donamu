# Donamu Project

## 프로젝트 소개
Donamu 프로젝트는 사내 구성원들의 업무 참여도와 성과를 시각적으로 확인할 수 있도록 설계된 내부 서비스이다.  
미션 기반 경험치 시스템과 랭킹 기능을 통해 구성원들의 참여를 유도하고 동기부여를 제공하는 것을 목표로 한다.

귀여운 UI 디자인과 게이미피케이션 요소를 적용하여 사용자들이 자연스럽게 미션에 참여하고 성취감을 느낄 수 있도록 구성하였다. 또한 개인 및 부서 단위의 랭킹 시스템을 통해 구성원 간 긍정적인 경쟁 구조를 형성하도록 설계하였다.

---

## 서비스 컨셉
- 귀여운 디자인과 게이미피케이션 요소를 활용한 사용자 참여 유도
- 랭킹 시스템을 통한 동기 부여 및 경쟁 구조 형성
- 미션 및 성과를 직관적으로 확인할 수 있는 UI 구성

---

## 기획 의도
서비스는 다음 세 가지 요소를 중심으로 설계되었다.

- 이용 편리성  
- 성과 가시성  
- 동기부여

사용자가 자신의 경험치와 성과를 쉽게 확인하고, 다른 구성원과 비교할 수 있도록 하여 자연스럽게 업무 참여를 유도하는 것을 목표로 한다.

---

## 주요 특징
- 재미 요소를 고려한 게이미피케이션 기반 구조
- 주 단위 / 월 단위 미션을 일정처럼 확인할 수 있는 UI
- 개인 경험치 및 부서 랭킹 시스템 제공
- 사용자 성과를 시각적으로 표현하여 몰입도 향상

---

## 시스템 아키텍처

### Application Layer
- React.js
- Node.js

### Backend Layer
- Spring Boot  
  - 서버 구현을 위한 핵심 프레임워크
- Spring Security  
  - JWT 토큰 기반 인증 및 권한 관리

### Database Layer
- MySQL  
  - 사용자 정보 및 서비스 데이터를 저장

### Notification Service
- Firebase  
  - Push 알림 기능 구현

### External Service
- Google API  
  - Google Spreadsheet와 연동하여 데이터 입력 및 수정 기능 제공

---

## 주요 기능

### 1. 로그인
- 아이디와 비밀번호 입력을 통한 사용자 인증

### 2. 나의 정보
- 사용자 기본 정보 조회
  - 사번
  - 소속
  - 이름
  - 캐릭터
  - 입사일
  - 레벨
- 캐릭터 수정
- 비밀번호 변경

### 3. 경험치 현황
- 누적 경험치 확인
- 현재 레벨 및 다음 레벨까지 필요한 경험치 표시
- 올해 획득 경험치 확인
- 카테고리별 경험치 확인

### 4. 우리팀 퀘스트
- 월별 / 주차별 미션 확인
- 직무별 퀘스트 및 리더 부여 퀘스트 관리
- 목표 달성 단계별 색상 구분 표시

### 5. 경험치 목록
- 최근 획득 경험치 확인
- 전체 경험치 목록 조회
- 연도별 경험치 기록 확인

### 6. 게시판
- 공지글 조회
- 공지글 검색
- 공지글 상세 조회

### 7. 알림 기능
- Push 알림 조회
- 공지 알림 및 경험치 알림 분류
- 알림 클릭 시 해당 화면 이동

---

## 관리자 기능

### 게시판 관리
- 공지글 조회
- 공지글 등록
- 공지글 수정
- 공지글 삭제

공지글 관리 기능을 통해 관리자가 효율적으로 공지사항을 관리할 수 있도록 설계하였다.

### 계정 관리
- 사용자 계정 조회
- 계정 생성
- 계정 수정
- 계정 삭제

기존 엑셀 기반 관리 방식보다 접근성과 편의성을 개선하였다.

---

## 랭킹 시스템

### 부서 랭킹
- 소속 부서의 등수 확인
- 앞뒤 등수 비교 가능

### 개인 랭킹
- 개인 경험치 순위 확인
- 상위 3명 표시
- 전체 랭킹 상위 100명 확인

랭킹 시스템을 통해 구성원들이 자신의 위치를 확인하고 자연스럽게 업무 참여 동기를 가질 수 있도록 설계하였다.

---
